## ✨ Overview

QR Generator is a lightweight web application that allows users to create QR codes instantly from text or URLs. Built with a clean and responsive interface, the project focuses on simplicity, usability, and fast QR code generation without requiring installation or registration.

---

## Highlights

- 📱 Generate QR codes instantly
- 🔗 Supports text and URL input
- ⚡ Fast and lightweight
- 💾 Download generated QR codes
- 📱 Responsive on desktop, tablet, and mobile
- 🎨 Clean and modern interface
